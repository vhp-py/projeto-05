# 📝 Gerenciador de Tarefas (CLI)

## 🎯 Sobre o Projeto
Este é um aplicativo de linha de comando (CLI) desenvolvido em Python para o gerenciamento prático de tarefas diárias (To-Do List). O sistema opera inteiramente no terminal e foi estruturado aplicando conceitos fundamentais de lógica de programação, laços de repetição e manipulação de listas e dicionários.

## ⚙️ Funcionalidades
O script oferece um painel interativo com as seguintes operações (CRUD):
* **Adicionar:** Insere novas tarefas na lista.
* **Visualizar:** Exibe todas as tarefas com indicadores visuais de status (pendente ou concluída `[✓]`).
* **Atualizar:** Permite renomear o texto de uma tarefa existente através do seu índice.
* **Completar:** Altera o status de uma tarefa específica para concluída.
* **Deletar:** Varre a lista e remove com segurança apenas as tarefas que já foram finalizadas.
